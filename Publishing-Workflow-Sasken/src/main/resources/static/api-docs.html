Base URL

http://localhost:8080/api

All API endpoints are relative to this base URL

Authentication

Currently, the API uses a simplified authentication model with user IDs.

All endpoints are publicly accessible for development purposes.

Available Endpoints

Blog Posts Management

POST /api/posts Create a new blog post
GET /api/posts Get all blog posts
GET /api/posts/{id} Get a specific blog post
PUT /api/posts/{id} Update a blog post
PUT /api/posts/{id}/status Change post status
DELETE /api/posts/{id} Delete a blog post

Monitoring & Statistics

GET /actuator/blogcraft/stats Get blog statistics
GET /actuator/blogcraft/health Get application health
GET /actuator/health System health check
GET /actuator/metrics Performance metrics

HTTP Response Codes

200 OK - Request successful
201 Created - Resource created successfully
400 Bad Request - Invalid input data
404 Not Found - Resource not found
500 Internal Server Error - Server error
